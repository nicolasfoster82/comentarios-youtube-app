# Landing Para Google Cloud

Esta carpeta contiene la landing estatica lista para publicar en:

`https://www.iniciativas.behealthpr.com/behealth-pr-comentarios/`

## Estructura

- `behealth-pr-comentarios/index.html`
- `behealth-pr-comentarios/politica-de-privacidad/index.html`
- `behealth-pr-comentarios/condiciones-del-servicio/index.html`
- `behealth-pr-comentarios/assets/site.css`

## Publicacion

Subi el contenido de este proyecto al directorio del subdominio que resuelve en:

`https://www.iniciativas.behealthpr.com/behealth-pr-comentarios/`

## Configuracion en Google Cloud

- `App name`: `BeHealth PR Comentarios`
- `Homepage URL`: `https://www.iniciativas.behealthpr.com/behealth-pr-comentarios/`
- `Privacy Policy URL`: `https://www.iniciativas.behealthpr.com/behealth-pr-comentarios/politica-de-privacidad/`
- `Terms of Service URL`: `https://www.iniciativas.behealthpr.com/behealth-pr-comentarios/condiciones-del-servicio/`

El nombre de la aplicacion debe coincidir exactamente en Google Cloud, la homepage, la politica de privacidad, las condiciones del servicio y cualquier interfaz visible del plugin.

La `Authorized redirect URI` no sale de esta landing. Debe seguir siendo la URL exacta que usa el plugin WordPress donde corre el OAuth.
