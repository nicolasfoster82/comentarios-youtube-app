# Landing Para Google Cloud

Esta carpeta contiene la landing estatica lista para publicar en:

`https://www.iniciativas.behealthpr.com/comentarios-youtube/`

## Estructura

- `comentarios-youtube/index.html`
- `comentarios-youtube/politica-de-privacidad/index.html`
- `comentarios-youtube/condiciones-del-servicio/index.html`
- `comentarios-youtube/assets/site.css`

## Publicacion

Subi el contenido de `landing/comentarios-youtube/` al directorio del subdominio que resuelve en:

`https://www.iniciativas.behealthpr.com/comentarios-youtube/`

## Configuracion en Google Cloud

- `App name`: `Comentarios de YouTube de BeHealth PR`
- `Homepage URL`: `https://www.iniciativas.behealthpr.com/comentarios-youtube/`
- `Privacy Policy URL`: `https://www.iniciativas.behealthpr.com/comentarios-youtube/politica-de-privacidad/`
- `Terms of Service URL`: `https://www.iniciativas.behealthpr.com/comentarios-youtube/condiciones-del-servicio/`

La `Authorized redirect URI` no sale de esta landing. Debe seguir siendo la URL exacta que usa el plugin WordPress donde corre el OAuth.
