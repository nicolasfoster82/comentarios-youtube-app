BeHealth PR Comentarios

BeHealth PR Comentarios es una funcionalidad implementada mediante un plugin de WordPress en este sitio. Permite que los visitantes inicien sesión con su cuenta de Google para publicar comentarios en videos de YouTube embebidos dentro de BeHealth PR.

BeHealth PR Comentarios es una aplicación propia de BeHealth PR e independiente de Google y YouTube. No está afiliada, patrocinada ni respaldada por esas empresas.

Autenticación

Conexión autorizada

La app usa el flujo de autorización de Google únicamente para autenticar al usuario y solicitar el permiso necesario para comentar en YouTube.

Uso del acceso

Comentarios en YouTube

El acceso solicitado se utiliza exclusivamente para permitir la publicación de comentarios en YouTube desde este sitio.

Datos sensibles

No guarda tu contraseña

Este sitio no almacena la contraseña de Google del usuario. La autenticación ocurre en los servicios de Google.

Propósito de la app

Qué hace y qué no hace esta integración

Esta página está pensada para que usuarios y revisores entiendan con claridad por qué la app solicita acceso con Google y cómo se usa ese acceso dentro del sitio.

01

Muestra videos embebidos

El plugin incrusta videos de YouTube dentro de páginas de WordPress para acompañar contenidos editoriales o institucionales.

02

Permite comentar

Si el usuario lo desea, puede iniciar sesión con Google y autorizar el acceso necesario para publicar comentarios en YouTube.

03

No gestiona otras acciones

La app no se usa para administrar canales, subir videos ni manipular contenido fuera de la función puntual de comentarios.

Flujo de uso

Cómo funciona para el usuario

El proceso es simple, voluntario y limitado a la publicación de comentarios en videos de YouTube embebidos en el sitio.

Inicia sesión con Google

El usuario toca el botón de conexión desde el reproductor y completa la autenticación directamente en Google.

Autoriza el permiso

Google muestra la pantalla de consentimiento para que el usuario apruebe el acceso necesario para comentar en YouTube.

Publica el comentario

Una vez autenticado, el usuario puede escribir y enviar un comentario para el video visible dentro del sitio.

El acceso puede revocarse en cualquier momento desde la cuenta de Google del usuario.
